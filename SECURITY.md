# Security Policy

## Supported Versions

| Version | Supported          | What this includes |
| ------- | ------------------ | ------------------ |
| 2.3     | :white_check_mark: | Current elooly.com |
| 2.1-2.2 | :x:                | Former elooly.com  |
| < 2.0   | :white_check_mark: | classic.elooly.com |

## Reporting a Vulnerability

If you see a vulnerability, please submit a [github issue](https://github.com/elooly10/elooly/issues/new) with the _Security_ tag
so we can know immediate attention is needed.
